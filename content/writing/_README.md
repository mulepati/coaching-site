# Adding Writing

To add a new poem or writing piece:

1. Copy `_template.md`.
2. Rename the copy to a URL-friendly filename, like `my-new-poem.md`.
3. Edit the metadata between the `---` lines.
4. Put the poem or writing below the second `---`.
5. Save the file and run `npm run dev`.

The filename becomes the page URL. For example:

`my-new-poem.md` becomes `/writing/my-new-poem`.

Use `order` to control where it appears on the Writing page. Lower numbers show first.
